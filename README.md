# finance-dashboard-web
A simple finance dashboard to track income, expenses, categories, upcoming bills and events using JavaScript and localStorage.


🌐 Live Demo
finflowprotracker.netlify.app


✨ Features
Dashboard - Real-time financial overview with income, expense, and balance tracking

Transactions - Add, edit, delete, search, and sort transactions

Upcoming Bills - Track pending bills with due date alerts

Financial Statements - Generate statements for any date range

Category Management - Custom categories for income, expenses, and bills

Event Planning - Create events and track event-specific expenses

Dark/Light Theme - Toggle between themes with persistent preference


🛠️ Technologies
HTML5, CSS3, JavaScript (Vanilla)

Font Awesome Icons

Google Fonts (Inter, Poppins)

LocalStorage for data persistence


🚀 Quick Start
Clone the repository

Open index.html in your browser

Start tracking your finances!

No build process or dependencies required.

📱 Responsive Design
Works on desktop, tablet, and mobile devices.
